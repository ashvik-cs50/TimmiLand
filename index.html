What is a "Game Nap" (Reboot)?

To make sure the voxel trees, funny cows, and fluffy sheep load as quickly as possible, TimmiLand builds the world right in your computer's high-speed memory lanes!

Sometimes, if you build a castle so huge that it goes all the way into the stars, the game engine gets super sleepy and takes a quick **2-second nap** to wipe the dust off! We call this a **Snooze Restart**:
